## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Bootstrap 5), JavaScript.
* **Herramientas de Construcción:** Vite (Entorno rápido y moderno).
* **Integraciones:** EmailJS (Envío de correos sin backend propio) y Google reCAPTCHA v2 (Seguridad activa).
* **Despliegue:** CI/CD mediante Netlify conectado a GitHub.

## ✨ Características Principales

- **Seguridad Avanzada:** Validación de datos en frontend y protección anti-spam profesional.
- **Optimización SEO:** Indexación técnica para motores de búsqueda y Google Analytics integrado.
- **Casos de Éxito:** Sección dedicada a proyectos reales como **Metrelex.cl**.

## ⚙️ Configuración Local
1. Clonar el repositorio.
2. Instalar dependencias: `npm install`.
3. Configurar variables de entorno: Crear un archivo `.env` con las siguientes llaves (excluidas del repositorio por seguridad):
   ```env
   VITE_EMAILJS_SERVICE_ID=tu_service_id
   VITE_EMAILJS_TEMPLATE_ID=tu_template_id
   VITE_EMAILJS_PUBLIC_KEY=tu_public_key
   VITE_RECAPTCHA_SITE_KEY=tu_site_key
   ```
4. Ejecutar en modo desarrollo: `npm run dev`.
5. Desplegar: El proyecto se despliega automáticamente a Netlify al hacer push al repositorio.
